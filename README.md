# 2020년도 기업사회맞춤형프로젝트2_01 RUStudent?
2020 2학기 동국대학교 기업사회맞춤형프로젝트 
유학생을 위한 학내 식당, 카페 Wechat 결제 미니프로그램 구현

## 개발 목적
Wechat MiniProgram을 이용하여 유학생의 학내 식당, 카페에서의 간편 주문 + Wechat Pay 결제 기능 제공.

## 개발 도구
- Wechat MiniProgram Developer Tools 
- Node.js 
- Express

## RUstudent 팀원
- [x] 서예현
- [x] 배성규
- [x] 정지영
- [x] 우승제
- [x] 황채환
